# Relocate Sprint Contract Directory Implementation Plan

> **For agentic workers:** Use team-driven-development to execute this plan.

**Goal:** Move Sprint Contract output from repo-root `sprints/` to `docs/team-dd/sprints/`, relocate existing directories, and update all references.
**Architecture:** Two sequential changes: first a `git mv` of six existing per-feature directories with a root-dir cleanup, then a cross-file string-substitution pass updating nine documentation / skill / agent files. Historical plan headers are intentionally skipped.
**Tech Stack:** Markdown, git, shell (bash).
**Spec:** docs/team-dd/specs/2026-04-18-relocate-sprints-dir-design.md (authoritative; consult for rationale/decisions)
**Sprints:** sprints/2026-04-18-relocate-sprints-dir/ (Sprint Contract files, generated by sprint-master)
