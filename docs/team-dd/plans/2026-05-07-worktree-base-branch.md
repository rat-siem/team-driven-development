# Worktree Base Branch Selection Implementation Plan

> **For agentic workers:** Use team-driven-development to execute this plan.

**Goal:** Add base-branch resolution + recording across the four skills, and replace `isolation: "worktree"` with Lead-managed `git worktree add` so Worker worktrees default to the current branch.
**Architecture:** Markdown-only edits to four SKILL files, two agent prompts, and one Worker prompt template. Verification is grep-based on file contents. No runtime tests.
**Tech Stack:** Git, Markdown, Bash
**Spec:** docs/team-dd/specs/2026-05-07-worktree-base-branch-design.md (authoritative; consult for rationale/decisions)
**Sprints:** docs/team-dd/sprints/2026-05-07-worktree-base-branch/ (Sprint Contract files, generated by sprint-master)
**Base branch:** develop
